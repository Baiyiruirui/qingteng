# 2026-07-15 · 今日案头返回与朝代标签修复

## 现象

- `/chat` 只在当前会话没有用户消息时展示今日案头；一旦进入聊天，主页面在该会话中没有返回入口。
- 诗笺地图曾出现英文 `Tang` 标签，虽然源数据和当前 Neon 数据验收均已是中文。

## 修复

- `/chat` 现在默认进入今日案头；发消息后切到对话视图，已有聊天时可用“继续对话 / 返回案头”双向切换。
- 旧会话 `/session/[conversationId]` 仍保持纯对话视图，不错误展示今日推荐。
- 新建对话后返回 `/chat` 的案头视图。
- 新增 `normalizeDynasty`,在诗笺服务端首屏、搜索 API 和今日诗笺三处把遗留 `Tang` 统一显示为“唐”。不修改数据库。

## 验证

- `pnpm verify:data`：通过；140 首数据库朝代值全部为中文。
- `pnpm build`：通过。
- `pnpm exec tsc --noEmit`：通过。
- `pnpm test:e2e`：公开入口 1 passed；登录后用例因未配置专用 E2E 账号跳过，并已补入案头/对话往返断言。
