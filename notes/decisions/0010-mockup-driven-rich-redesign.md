# 0010. 按示意图重设计:今日案头 + 书法字体 + 意境图管线

**日期**: 2026-07-06
**状态**: Accepted

## 背景

Owner 认为现有页面"太素",给出三张高浓度水墨示意图(首页案头/诗笺书房/诗境长卷)。
拆解结论:浓度差距 = 大幅水墨插画(~50%) + 信息密度(~25%) + 书法大标题(~15%) + 质感细节(~10%)。
其中信息密度部分的模块(札记/今日入诗/青藤记得/归类/体会进度)**大半能直连已有后端**。

## 决策(Owner 三连拍板)

1. **按示意图完整改**(视觉 + 信息架构一起),不只是换皮
2. **意境图走"Owner 生图 → Claude Code 接入"管线**,prompt 见 `notes/design-image-prompts.md`(5 张:垂藤/远山/静夜思/九月九/登高长卷),放 `public/yijing/`
3. **从首页(聊天开场)开始**,依次诗库、沉浸页

## 首页方案:「今日案头」= 聊天页的空状态

关键设计判断:示意图 1 的仪表盘不另起路由,而是作为 `/chat` **无对话时的空状态**。
从案头输入框发第一句话 → 自然切换为对话视图。案头三栏:

| 模块 | 数据源 |
|---|---|
| 今日青藤札记 | 开场白 API(原本进气泡,现进札记) |
| 今日入诗(轻量推荐) | 有沉浸脚本的诗按日轮换 + 画像 themes 出推荐理由 |
| 青藤记得(最近读过/常聊意象/记在心里/下一步) | `/api/memory/context` |
| 读一首/练一题/写两句 | 三模式入口(诗库/quiz/creative) |

## 字体

新增 **Ma Shan Zheng(马善政楷体)** via next/font,token `--font-kai` / 类名 `font-kai`。
**约束:只用于大标题/牌匾式文字,正文禁用**(书法体可读性差)。
备选已查证:Long Cang、Zhi Mang Xing 也在 Google Fonts。

## 后果

**好的**:后端能力(三层 Memory/开场白/画像)第一次完整可见;首屏叙事"它记得你"成立。
**坏的**:Ma Shan Zheng 全量 CJK 字体较大(preload:false 缓解;Phase D 字体子集化时一并优化)。
**代价**:意境图依赖 Owner 生图,未到位前浓度只有代码层的七成。

## 如果反悔了

- 案头若干扰"直接聊天"的用户,可加"跳过案头"偏好或首条消息直达。
- 书法字体若在低端机渲染差,退回 Noto Serif SC 加大字重。
