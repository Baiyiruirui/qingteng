# 青藤 · 设计系统规范(DESIGN_SYSTEM.md)

> 这是青藤的视觉地基。所有页面、组件统一遵循。Claude Code 实施美术时严格按此执行。
> 风格定调:**宣纸水墨为主,暖色极克制点缀**。锁定亮色模式,不做暗色。

## 2. 字体 · 排版

### 引入 Noto Serif SC(中文衬线)
- 通过 Google Fonts 或 self-host 引入 `Noto Serif SC`(weights: 400, 500)
- Next.js 用 `next/font/google` 引入,避免 FOUT,优化加载
- 中文标题、"青藤"二字、诗句、副标题 → Noto Serif SC
- 正文、说明、按钮内文字、表单 → 系统 sans(现有)

### 字号层级
| 用途 | 字号 | 字重 | 字体 |
|---|---|---|---|
| 主标题(青藤) | 28-32px | 500 | Serif |
| 页面标题 | 22px | 500 | Serif |
| 诗句 / 副标题 | 17-19px | 400 | Serif |
| 正文 | 14-15px | 400 | Sans |
| 提示 / 小字 | 12-13px | 400 | Sans |

标题/诗句加 `letter-spacing: 2-4px`(衬线中文加字距更雅)。

### 按钮
- 主按钮:墨色底 `#3A4742` + 宣纸字 `#F2EDE0`,serif,letter-spacing 3px,圆角 8px
- 次按钮:透明底 + 中墨字 + 0.5px 暖灰边框
- 链接:朱砂 `#C0623F`

## 4. 动效语言(慢柔墨晕感)

水墨产品的动效要"慢、柔、像墨在纸上晕开"。**禁止科技产品那种弹跳/快闪/缩放。**

| 规则 | 值 |
|---|---|
| 时长 | 0.4-0.6s(比常规慢) |
| 缓动 | `ease-out` 或 `cubic-bezier(0.22, 1, 0.36, 1)`(柔和) |
| 入场 | 以淡入(opacity)+ 轻微上移(translateY 8-12px)为主,模拟墨晕开 |
| 禁止 | 弹跳(bounce)、快速缩放、闪烁、旋转、生硬位移 |

- 页面/卡片进入:淡入 + 轻上移
- 消息流式:逐字本身就是动效,不额外加
- hover:颜色/透明度的柔和过渡,不要位移跳动
- 用 Framer Motion 时统一封装一个 `inkFadeIn` variant 复用

## 5. 宣纸纹理质感层

纯色平涂缺质感。加一层**极淡的宣纸纤维/噪点纹理**叠在底色上。

- 用 SVG `feTurbulence` 噪点滤镜生成,或一张极淡的纹理 PNG(self-host)
- **透明度极低**(opacity 3-6%),几乎看不见,但能感觉到"纸"
- 固定在页面最底层(`position: fixed; z-index: -1; pointer-events: none`)
- 不能影响文字可读性,不能拖慢加载(SVG 噪点优先,无需额外请求)

参考实现(SVG 噪点,内联):
```html
<svg style="position:fixed;inset:0;z-index:-1;width:100%;height:100%;opacity:0.04;pointer-events:none">
  <filter id="paper-noise"><feTurbulence type="fractalNoise" baseFrequency="0.8" numOctaves="2"/></filter>
  <rect width="100%" height="100%" filter="url(#paper-noise)"/>
</svg>
```

## 6. 标志性符号 · 朱砂印章

青藤的视觉记忆点 = **一枚朱砂"青藤"印章**。反复出现形成记忆:
- Logo 旁(已有雏形)
- 加载/空状态
- 做题全对、完成沉浸等"成就时刻"盖一个印
- 印章样式:朱砂 `#C0623F`,方形篆刻感,可用 SVG 绘制("青藤"或单字"藤")
- 做一个可复用的 `<SealStamp>` 组件,不同场景调用

## 7. 诗意空状态 / 加载

不要敷衍的"暂无数据"。用诗意化设计:

| 场景 | 设计 |
|---|---|
| 错题本为空 | 一笔淡墨 + "暂无错题,如清池无尘" |
| 对话空状态 | 已有"和青藤说说话吧",保留并配淡墨意向 |
| 加载中 | 一笔墨慢慢晕开的动画(SVG path + 动效),或淡淡的"青藤"印章渐显 |
| 诗库无搜索结果 | "未寻得此诗,换个词试试?" |

文案都要有诗意、温度,不用冷冰冰的系统语。
